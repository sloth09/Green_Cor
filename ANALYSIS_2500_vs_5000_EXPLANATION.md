# 2500m³ vs 5000m³ 셔틀 비용 분석
## Case 1: 왜 20년 최적화와 2030년 최적값이 다를까?

---

### 최적화 결과의 패러독스

| 관점 | 최적 선택 | NPC 총 비용 | 필요 셔틀 수 |
|------|---------|-----------|----------|
| **2030년 (첫 해만)** | 5000m³ | $14.92M | 1척 |
| **20년 누적 (2030-2050)** | 2500m³ | $106.90M | 11척 |

**차이: $9.03M (7.8%)**
- 2500m³이 20년 동안 $9.03M 더 저렴
- 그런데 2030년만 보면 5000m³이 $3.55M 더 저렴

## 🔍 원인 분석: 3단계

### **1️⃣ 2030년 (첫 해) - 왜 5000m³이 더 저렴해 보이는가?**

```
2500m³ 비용: $18.47M
├─ CAPEX (초기 셔틀 구매):  $17.11M  ← 2척 × $8.56M/척
└─ OPEX (운영비):           $1.37M

5000m³ 비용: $14.92M
├─ CAPEX (초기 셔틀 구매):  $13.89M  ← 1척 × $13.89M/척
└─ OPEX (운영비):           $1.03M

✅ 결과: 5000m³이 $3.55M 더 저렴
```

**이유:**
- **2500m³**: 5,000m³ 수요를 충족하려면 **2척 필요**
  - 초기 구매: 2척 × $8.56M = $17.11M
  - 운영비가 높음 (2척 유지)

- **5000m³**: 5,000m³ 수요를 충족하려면 **1척으로 충분**
  - 초기 구매: 1척 × $13.89M = $13.89M
  - 운영비가 낮음 (1척만 유지)

📌 **핵심**: 초기 연도는 **단위당 CAPEX 차이**가 절대값으로 크므로, 수량 × 단가의 효과가 중요

---

### **2️⃣ 누적 효과 (20년 동안) - 왜 2500m³이 역전되는가?**

```
필요 셔틀 수 변화 (수요 증가에 따라)

         2500m³ vs 5000m³ 비교
Year     2030  2035  2040  2045  2050
─────────────────────────────────────
2500m³:   2     4     5     8    11척
5000m³:   1     2     3     4     8척
차이:    +1   +2    +2    +4    +3척  ← 2500m³이 더 많은 셔틀 필요!
```

**누적 비용 구조:**

```
2500m³ 20년 누적 비용: $106.90M
├─ Shuttle CAPEX:      $49.99M  (11척 총 구매)
├─ Pump CAPEX:         $5.63M
├─ Fixed OPEX:         ~$29M    (유지보수, 보험 등)
└─ Variable OPEX:      ~$22M    (연료, 냉각 등)

5000m³ 20년 누적 비용: $115.93M
├─ Shuttle CAPEX:      $62.82M  (8척 총 구매, 단가↑)
├─ Pump CAPEX:         $4.66M
├─ Fixed OPEX:         ~$21M    (1척 적으니까 덜 들음)
└─ Variable OPEX:      ~$27M    (크기가 크니까 더 들 수 있음)
```

**핵심 트레이드오프:**

| 항목 | 2500m³ | 5000m³ | 영향 |
|-----|--------|--------|------|
| **총 Shuttle CAPEX** | $49.99M | $62.82M | ❌ 5000m³이 $12.83M 더 비쌈 |
| **Fixed OPEX** | ~$29M | ~$21M | ✅ 2500m³이 $8M 더 비쌈 (운영비) |
| **최종 결과** | $106.90M | $115.93M | ✅ 2500m³이 $9.03M 더 저렴 |

### **3️⃣ 전환점은 언제인가?**

그래프의 **상단 좌측** (누적 현재가)를 보면:

```
연도별 누적 비용 차이:

2030년:  5000m³이 $3.55M 낮음  ← 5000m³ 유리
2035년:  5000m³이 $14.0M 낮음  ← 여전히 5000m³ 유리
2040년:  거의 비슷  (~$0)      ← 교차점!
2045년:  2500m³이 $5M 낮음    ← 2500m³이 역전!
2050년:  2500m³이 $9.03M 낮음 ← 장기에는 2500m³ 유리

🔄 전환 원리:
  - 초기(2030-2035): 5000m³의 초기 CAPEX 아래로 내려감
  - 중반(2040년경): 두 방식의 누적 비용이 같음
  - 후기(2040-2050): 2500m³이 필요한 추가 셔틀 CAPEX와 OPEX가 누적되기 시작
               → 2500m³이 단위당 CAPEX는 낮지만, "필요한 수량"이 많아서
               → 결국 총 CAPEX와 OPEX가 더 누적됨
```

## 📈 그래프 해석

### **그래프 1: 누적 현재가 (Cumulative PV)**
```
가장 중요한 그래프!

Y축: 누적 할인된 비용 (낮을수록 좋음)
X축: 연도

파란선 (2500m³):   아래로 완만히 상승
주황선 (5000m³):   초기 높지만 상승률이 더 가파름

→ 20년 후: 파란선이 아래에 있음 = 2500m³이 더 저렴
```

### **그래프 2: 연간 CAPEX (Annual Capital Expenditure)**
```
2030년의 초기 구매 비용:
  - 2500m³: $17.11M (2척)
  - 5000m³: $13.89M (1척)

2033년, 2035년, 2040년 등에 추가 구매:
  - 2500m³: 더 빈번하게 추가 구매 필요
  - 5000m³: 덜 빈번하게 추가 구매

→ 누적되면 2500m³의 CAPEX가 더 커짐
```

### **그래프 3: 누적 셔틀 수 (Total Shuttles)**
```
가장 명확한 차이!

2030년:  2척 vs 1척  (2배 차이)
2050년:  11척 vs 8척 (37% 차이)

→ 수요가 증가할수록, 더 많은 셔틀이 필요한 방식의 누적 CAPEX가 증가
→ 2500m³ × 11척의 총 구매비 > 5000m³ × 8척의 총 구매비?
   아니다! 역설이다:
   - 2500m³은 단위당 저렴 ($8.56M/척)
   - 5000m³은 단위당 비쌈 ($7.85M/척)

   그런데 왜 최종적으로 2500m³이 더 저렴할까?
   → OPEX 효과 때문! (더 적은 셔틀 유지비)
```

### **그래프 4: 2030년 vs 2050년 비교**
```
막대 그래프로 명확하게 보여줌:

2030년 (단일 연도):
  - 2500m³: $18.5M
  - 5000m³: $14.9M  ← 5000m³이 저렴
  → 5000m³을 선택하면 첫 해에 $3.6M 절감!

2050년까지 (20년 누적):
  - 2500m³: $106.9M
  - 5000m³: $115.9M  ← 2500m³이 저렴
  → 2500m³을 선택하면 20년에 $9.0M 절감!
```

## 💡 왜 이런 역설이 발생하는가?

### **근본 원인: 규모의 경제 vs 초기 투자**

```
2500m³ 전략:
  ✅ 장점:
     - 초기 투자 낮음 (1척으로 시작)
     - 작은 셔틀이므로 단위 CAPEX 낮음

  ❌ 단점:
     - 수요 증가 시 "여러 척" 필요 (많은 수량)
     - 각 셔틀마다 OPEX 발생 (유지비, 보험, 냉각)
     - 운영 복잡도 증가 (더 많은 선박 관리)

5000m³ 전략:
  ✅ 장점:
     - 수요 충족을 위해 "적은 수의" 큰 셔틀 필요
     - OPEX가 낮음 (운영할 선박 수 적음)
     - 운영 단순성 (선박 수 적음)

  ❌ 단점:
     - 초기 투자 높음 (큰 셔틀은 비쌈)
     - 단위당 CAPEX 높음 (규모의 한계 효과)

🔑 핵심:
   초기 높은 투자(CAPEX) vs 누적된 운영비(OPEX)
   단기(2030년): CAPEX 절감이 중요 → 2500m³ 선호
   장기(2050년): 누적 OPEX 차이가 누적 → 5000m³이 역전당함
```

## 🔬 기술 노트: 왜 5000m³이 최종적으로 더 비싼가?

### **Shuttle Cost Scaling Law**

```
CAPEX = reference_capex × (size / reference_size)^alpha

Example:
  CAPEX(2500) = 61.5M × (2500/40000)^0.75 = $8.56M
  CAPEX(5000) = 61.5M × (5000/40000)^0.75 = $13.89M

⚠️ 중요: 크기가 2배 증가해도 비용은 1.62배만 증가 (부분적 규모 경제)

그런데:
  2500m³ × 11척 = $94.16M (Shuttle CAPEX)
  5000m³ × 8척  = $111.12M (Shuttle CAPEX)

  차이 = $16.96M 증가!

→ 비용의 비선형 증가: 규모의 경제가 있어도,
  수량 증가가 더 크면 총 비용은 증가
```

### **OPEX 효과**

```
Fixed OPEX (유지보수, 보험, 냉각):
  2500m³: $29M (11척 × 높은 개당 비용)
  5000m³: $21M (8척 × 낮은 개당 비용)

절감: $8M

Variable OPEX (연료, 에너지):
  크기 효과: 큰 셔틀일수록 에너지 효율 낮을 수 있음
  → 약간 부정적 효과

전체: 2500m³이 약 $8M 더 저렴한 OPEX
```

## 🎓 학술적 해석: 최적화 이론

### **다목적 최적화 (Multi-Objective Optimization)**

```
Objective 1: Minimize Initial CAPEX
  → 5000m³ wins ($13.89M vs $17.11M)

Objective 2: Minimize Total NPC over 20 years
  → 2500m³ wins ($106.90M vs $115.93M)

이는 "Pareto Frontier"의 전형적인 예시:
두 목표가 상충하며, 가중치에 따라 최적해가 달라짐
```

### **시간 관점의 할인 효과 (Discounting Effect)**

```
할인율 = 7% (현재 설정)

2030년 $1M = 2030년의 $1M
2050년 $1M = 2030년의 $0.26M (20년 후에는 25%로 축소됨)

→ 초기 투자가 상대적으로 "무겁게" 계산됨
→ 장기 운영비는 상대적으로 "가볍게" 계산됨

따라서:
  초기 CAPEX 차이 ($3.22M) vs 누적 OPEX 차이 ($11M)
  할인 적용 후: 초기 차이가 더 크게 작용
  → 이것이 2030년에는 5000m³이 유리하게 보이는 이유
```
