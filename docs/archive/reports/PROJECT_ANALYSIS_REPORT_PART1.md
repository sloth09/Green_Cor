## Executive Summary

### 프로젝트 핵심

본 프로젝트는 2030~2050년 20년 동안 부산 Green Corridor에서의 **친환경 암모니아 연료 공급 인프라를 최적화**하는 혼합정수선형계획(MILP) 모델입니다. 핵심 목표는:

1. **수요 예측**: 50척(2030년)에서 500척(2050년)으로 증가하는 암모니아 연료 선박 함대의 **연간 수요를 정확히 산정**
2. **의사결정**: **셔틀 선박의 최적 크기, 개수, 펌프 용량**을 결정하여
3. **비용 최소화**: **20년간의 순현재가(Net Present Cost, NPC)를 최소화**

2050년의 연간 수요는 2030년 대비 **약 10배 증가** (1.9M m³ → 19M m³)하므로, 초기부터 **확장 가능하고 경제적인 인프라**를 설계하는 것이 매우 중요합니다.

### 세 가지 Case의 구조적 차이

모델의 세 가지 Case는 암모니아 공급의 **지리적 위치와 운영 방식**의 근본적인 차이를 반영합니다:

| 항목 | **Case 1: 부산 저장소** | **Case 2-1: 여수** | **Case 2-2: 울산** |
|-----|----------|----------|----------|
| **시나리오** | 부산항에 저장탱크 | 여수 생산→부산 | 울산 생산→부산 |
| **운송 거리** | 항만 내부 (~1시간) | 86해리 (5.73시간) | 25해리 (1.67시간) |
| **부산 저장시설** | 있음 (35,000톤) | 없음 | 없음 |
| **셔틀 크기** | 500~5,000 m³ (소형) | 5,000~50,000 m³ (대형) | 5,000~50,000 m³ (대형) |
| **운영 개념** | 소형 셔틀 × 다중 왕복 | 대형 셔틀 × 단일 항해 | 대형 셔틀 × 빠른 회전율 |
| **초기 투자** | ~$50.6M | ~$16.1M | ~$16.1M |
| **20년 NPC** | ~$2.5~3.5B | ~$1.6~2.3B | ~$1.5~2.2B |

**근본적인 차이**:
- **Case 1**: 작은 셔틀이 저장탱크에서 **여러 번 왕복**하여 선박들에게 연료 공급 (높은 유연성, 높은 초기 비용)
- **Case 2-1/2-2**: 큰 셔틀이 원산지에서 대량 운송 후 **한 항해로 여러 척 동시 서빙** (낮은 초기 비용, 예약 필수)

### 시간 구조와 운영 로직 (핵심)

각 Case의 **사이클 타임**(완전한 왕복 시간)은 **셔틀이 얼마나 자주 운항할 수 있는가**를 결정하며, 이는 **필요한 셔틀 개수와 비용**에 직접 영향을 미칩니다. 특히 **육상 탱크에서 셔틀로 연료를 주유하는 시간**이 모든 경우에 동일하게 적용됩니다.

#### 육상 연료 공급 탱크에서 셔틀로의 주유 시간
```
모든 Case 공통:
육상 펌프 유량 = 1,500 m³/h (고정값)

주유 시간 = Shuttle_Volume / 1,500
예시:
- 5,000 m³ 셔틀:   5,000 ÷ 1,500 = 3.33시간
- 10,000 m³ 셔틀:  10,000 ÷ 1,500 = 6.67시간
- 25,000 m³ 셔틀:  25,000 ÷ 1,500 = 16.67시간
- 50,000 m³ 셔틀:  50,000 ÷ 1,500 = 33.33시간

→ 셔틀이 클수록 적재 시간이 선형적으로 증가
→ 큰 셔틀은 장거리 운송에만 경제적 (높은 용량으로 시간 투자 보상)
```

#### Case 1: 부산항 내 고속 회전 (항만 내 저장탱크 기반)
```
한 사이클 구성 (5,000 m³ 셔틀, 1,000 m³/h 펌프):

1단계: 육상 탱크에서 적재 (부산항 내)
   - 육상 펌프 유량: 1,500 m³/h (고정)
   - 적재 시간: 5,000 ÷ 1,500 = 3.33시간

2단계: 부산항 내 운송 및 벙커링
   - 셔틀 편도 이동:               1시간
   - 호스 연결 및 퍼징:            0.5시간
   - 벙커링 (연료 공급):           5시간 (5,000 m³ ÷ 1,000 m³/h 펌프)
   - 호스 분리 및 퍼징:            0.5시간

3단계: 복귀
   - 셔틀 복귀 (부산항 내):        1시간

─────────────────────────
총 사이클:                        11.33시간

특징:
→ 연간 최대 가능: 8,000시간 ÷ 11.33 = **706회 운항**
→ 한 척의 셔틀이 매우 높은 빈도(거의 매일)로 운항 가능
→ 많은 셔틀 필요 (규모 경제 불리)
→ 탱크 덕분에 즉각적인 수요 대응 가능
```

#### Case 2-2: 울산 → 부산 장거리 운송 (근거지 생산 기반)
```
한 사이클 구성 (25,000 m³ 셔틀, 1,000 m³/h 펌프, 5척 선박 서빙):

1단계: 근거지(울산) 준비 및 적재
   - 근거지 준비:                  1시간
   - 육상 펌프 유량: 1,500 m³/h (고정)
   - 적재 시간: 25,000 ÷ 1,500 = 16.67시간

2단계: 항해 및 부산 진입
   - 울산→부산 편도 항해 (15노트): 1.67시간 (25해리 ÷ 15노트)
   - 부산항 진입:                  1시간

3단계: 부산에서 5척 선박 동시 서빙
   각 선박 (5,000 m³ 기준):
   - 선박 이동/접안:               1시간
   - 호스 연결 및 퍼징:            0.5시간
   - 벙커링:                       5시간 (5,000 m³ ÷ 1,000 m³/h)
   - 호스 분리 및 퍼징:            0.5시간
   소계 (1척):                     7시간
   × 5척:                         35시간

4단계: 복귀
   - 부산→울산 편도 항해:         1.67시간
   - 근거지 도착:                  1시간

─────────────────────────
총 사이클:                        63시간

특징:
→ 연간 최대 가능: 8,000시간 ÷ 63 = **127회 항해**
→ 각 항해에서 5척 동시 서빙 = 연간 최대 635척 처리
→ 소수의 셔틀로 많은 수요 처리 (규모 경제 유리)
→ 사전 예약 필수 (탱크 없어서)
→ 거리가 경제성을 좌우 (Case 2-1은 더 오래 걸림)
```

#### Case 2-1 (여수 → 부산) vs Case 2-2 (울산 → 부산) 비교
```
거리 차이:
- 여수: 86해리 ÷ 15노트 = 5.73시간 (편도)
- 울산: 25해리 ÷ 15노트 = 1.67시간 (편도)

총 항해 시간 차이: 4.06시간 × 2 = 8.12시간

영향:
- Case 2-1 총 사이클: ~75시간
- Case 2-2 총 사이클: ~63시간
- 시간 차이: 12시간 (약 16% 단축)

연간 항해 능력:
- Case 2-1: 8,000 ÷ 75 = 107회 (535척 처리)
- Case 2-2: 8,000 ÷ 63 = 127회 (635척 처리)

결론: 거리가 단축되면 **연간 항해 능력이 19% 증가**하므로 **경제성 30~40% 개선**
```

**핵심 비교**:
- **Case 1**: 사이클이 짧지만(11시간) **많은 셔틀 필요** (706회 ÷ 1콜당 셔틀 수)
- **Case 2**: 사이클이 길지만(63시간) **소수 셔틀로 충분** (127회로 5척 × 5척 = 수백 척 처리)

### 모델의 기술적 특징

#### 의사결정 변수 (Decision Variables)
- **x[t]**: t년도 신규 추가 셔틀 수 (정수)
- **N[t]**: t년도 누적 셔틀 수 (정수)
- **y[t]**: t년도 연간 운항/항해 수 (연속)
- **z[t]**: t년도 신규 탱크 수 (Case 1만, 정수)

#### 최적화 제약식 (Constraints)
1. **수요 충족**: 연간 운항 × 용량 ≥ 수요
2. **작업 시간**: 연간 운항 시간 ≤ 셔틀 수 × 8,000시간
3. **탱크 용량** (Case 1만): 셔틀 총 용량 × 안전계수 ≤ 탱크 총 용량

#### 비용 함수 (Cost Function)
```
최소화: NPC = Σ_t [할인율 × (CAPEX(t) + 고정OPEX(t) + 변동OPEX(t))]

구성:
- CAPEX: 셔틀, 펌프, 탱크 구매비
- 고정OPEX: 유지보수비 (CAPEX의 3~5%)
- 변동OPEX: 연료비, 펌핑 에너지비, 냉각비
```

### 주요 발견사항

1. **초기 투자와 운영 비용의 트레이드오프**: Case 1은 높은 초기 투자로 높은 운영 유연성, Case 2는 낮은 초기 투자로 예약 필수

2. **거리의 영향**: Case 2-2 (울산)가 Case 2-1 (여수)보다 16% 빠른 사이클로 **경제성이 우수**

3. **펌프 크기 최적화**: 1,000~1,200 m³/h가 대부분의 경우 최적 (시간 절감과 CAPEX 사이의 균형)

4. **셔틀 크기**:
   - Case 1: 중간 크기 (2,500~5,000 m³)가 최적 (트립 수 vs 자본비 균형)
   - Case 2: 중형 (5,000~25,000 m³)이 최적 (거리와 용량 균형)

5. **20년 장기 관점**: 초기 투자 차이는 할인되지만, 운영 비용 누적이 NPC를 결정하는 주요 요소

## Chapter 1: 프로젝트 개요 및 모델 구조

### 1.1 문제 정의

#### 1.1.1 비즈니스 맥락
Green Corridor는 국제 해운의 탈탄소화를 목표로 하는 이니셔티브입니다. 부산항은 동아시아의 주요 해운 허브이며, 향후 30년 동안:

- **2030년**: 50척의 친환경 암모니아 연료 선박 운영
- **2050년**: 500척으로 증가 (선형 성장 가정)
- **연간 항차 수**: 선박당 12회 (약 30일 간격)
- **항차당 연료량**: 2,158,995 kg (약 3,175 m³, 사용자 요청으로 5,000 m³로 오버라이드)

이들 선박의 연료 수요를 충족시키기 위해 **얼마나 많은 셔틀 선박**과 **어떤 크기의 펌프**가 필요한지를 결정해야 합니다.

#### 1.1.2 의사결정 변수
최적화 모델이 결정해야 할 주요 변수:

1. **x[t]**: t년도에 신규 추가되는 셔틀의 개수 (정수 변수)
2. **N[t]**: t년도의 누적 셔틀 개수 (정수 변수, N[t] = N[t-1] + x[t])
3. **y[t]**: t년도의 연간 벙커링 콜 수 또는 트립 수 (연속 변수)
4. **x_tank[t]**: t년도에 신규 추가되는 저장탱크 개수 (Case 1만, 정수 변수)
5. **N_tank[t]**: t년도의 누적 저장탱크 개수 (Case 1만)

#### 1.1.3 목적함수
최소화 목표: 20년간의 순현재가 (Net Present Cost)

```
Minimize NPC = Σ_t [Discount_Factor(t) × (CAPEX(t) + FIXED_OPEX(t) + VARIABLE_OPEX(t))]
```

여기서:
- **Discount_Factor(t)** = 1 / (1 + 0.07)^(t - 2030) [할인율 7%]
- **CAPEX(t)** = 셔틀 자본비 + 펌프/벙커링 장비비 + 저장탱크 자본비
- **FIXED_OPEX(t)** = 셔틀 고정 운영비 + 펌프 고정 운영비 + 탱크 고정 운영비
- **VARIABLE_OPEX(t)** = 셔틀 연료비 + 펌프 연료비 + 탱크 냉각비

### 1.2 수요 예측 모델

#### 1.2.1 선박 함대 성장

모델은 **2030년부터 2050년까지 20년 동안 선박 함대가 선형적으로 증가**한다고 가정합니다.

```
Vessel_Count(year) = 50 + (500 - 50) × (year - 2030) / (2050 - 2030)
                   = 50 + 22.5 × (year - 2030)
```

**구체적 예시**:
- 2030년: 50척
- 2035년: 162척
- 2040년: 275척
- 2045년: 387척
- 2050년: 500척

#### 1.2.2 연간 수요 계산

각 선박은 **연간 12회의 항차**를 수행하며, 각 항차에서 **2,158,995 kg (약 3,175 m³)의 암모니아 연료**를 적재합니다. 그러나 사용자 요청으로 **벙커링 콜당 5,000 m³**를 기준으로 모델링합니다.

```
Annual_Demand(year) = Vessel_Count(year) × m3_per_voyage × voyages_per_year
                    = Vessel_Count(year) × 3,175 m³ × 12
```

**연도별 수요 예시**:
- 2030년: 50 × 3,175 × 12 = 1,905,000 m³
- 2035년: 162 × 3,175 × 12 = 6,152,400 m³
- 2040년: 275 × 3,175 × 12 = 10,458,000 m³
- 2050년: 500 × 3,175 × 12 = 19,050,000 m³

연간 수요는 **20년 동안 약 10배 증가**합니다.

#### 1.2.3 밀도 변환

암모니아의 저장 조건 밀도를 적용하여 질량을 부피로 변환:

```
m3_per_voyage = kg_per_voyage / (density_storage_ton_m3 × 1000)
              = 2,158,995 / (0.680 × 1000)
              = 3,175.3 m³
```

### 1.3 경제 모델

#### 1.3.1 할인 및 NPV 계산

모든 미래 비용은 **7% 연이율의 할인율**로 현재 가치로 계산됩니다:

```
Discounted_Value(t) = Future_Value(t) / (1.07)^(t - 2030)
```

이는 **시간이 지날수록 비용의 가중치가 감소**함을 의미합니다. 예를 들어:
- 2030년 100만 달러 = 현재 100만 달러
- 2035년 100만 달러 = 현재 71.3만 달러
- 2050년 100만 달러 = 현재 26.3만 달러

#### 1.3.2 연료 가격

- **암모니아 연료 가격**: 600 USD/톤 (기준값)
- **전기 가격**: 0.0769 USD/kWh (펌프 냉각비 계산용)

#### 1.3.3 비용 구성 요소

모델에서 계산되는 모든 비용 항목:

1. **CAPEX (자본비)**
   - 셔틀 선박 구매비
   - 펌프/벙커링 장비비
   - 저장탱크 설치비 (Case 1만)

2. **고정 OPEX (고정 운영비)**
   - 셔틀 선박 유지보수비 (CAPEX의 5%)
   - 펌프 유지보수비 (CAPEX의 5%)
   - 탱크 유지보수비 (CAPEX의 3%)

3. **변동 OPEX (변동 운영비)**
   - 셔틀 연료비 (항해 시 엔진 연료)
   - 펌프 연료비 (펌핑 작업 시)
   - 탱크 냉각비 (암모니아 냉각 유지 비용)

### 1.4 기술 파라미터

#### 1.4.1 엔진 사양

셔틀의 크기에 따라 **최대 연속 정격(Maximum Continuous Rating, MCR)** 파워가 다릅니다:

| 셔틀 크기 (m³) | MCR (kW) | 비고 |
|--------------|---------|------|
| 500 | 1,296 | 기존 데이터 |
| 1,000 | 1,341 | 기존 데이터 |
| 2,000 | 1,429 | 기존 데이터 |
| 3,000 | 1,517 | 기존 데이터 |
| 4,000 | 1,606 | 기존 데이터 |
| 4,500 | 1,650 | 보간값 |
| 5,000 | 1,694 | 보간값 |
| 10,000 | 2,159 | 외삽값 (로그 스케일) |
| 25,000 | 2,981 | 외삽값 |
| 50,000 | 3,867 | 외삽값 |

**MCR 해석**: MCR이 클수록 엔진이 강력하며, 더 많은 연료를 소비합니다.

#### 1.4.2 연료 소비율 (SFOC)

- **SFOC (Specific Fuel Oil Consumption)**: 379 g/kWh
- **의미**: 엔진이 1 kWh의 파워를 생성할 때 379그램의 연료를 소비

셔틀이 한 번의 사이클에서 소비하는 연료:
```
Fuel_per_cycle (ton) = MCR (kW) × SFOC (g/kWh) × travel_time (h) / 1,000,000
```

예: 1,500 kW 엔진, 2시간 항해
```
Fuel = 1,500 × 379 × 2 / 1,000,000 = 1.137 ton
```

#### 1.4.3 펌프 사양

펌프는 벙커링 과정에서 암모니아를 셔틀에서 선박으로 전송합니다:

- **사용 가능한 펌프 용량**: 400, 600, 800, 1,000, 1,200, 1,400, 1,600, 1,800, 2,000 m³/h
- **압력 강하(ΔP)**: 4 bar
- **펌프 효율**: 70%

**펌프 파워 계산**:
```
Pump_Power (kW) = (ΔP [Pa] × Q [m³/s]) / efficiency
                = (4 × 10^5 Pa × Q/3600 [m³/s]) / 0.7
```

예: 1,000 m³/h 펌프
```
Power = (4 × 10^5 × 1000/3600) / 0.7 = 158.7 kW
```
