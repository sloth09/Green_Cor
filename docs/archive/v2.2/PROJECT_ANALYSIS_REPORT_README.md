## 💡 주요 발견사항

### 1. 시간 구조의 중요성
```
육상 연료 공급 시간 (모든 Case 동일):
주유 시간 = Shuttle_Volume / 1,500 m³/h

예시:
- 5,000 m³ 셔틀: 3.33시간
- 25,000 m³ 셔틀: 16.67시간
```

### 2. Case별 사이클 타임
```
Case 1 (부산항 내 고속 회전):
- 총 사이클: 11.33시간
- 연간 최대 운항: 706회

Case 2-2 (울산 장거리):
- 총 사이클: 63시간
- 연간 최대 항해: 127회 (5척 동시 서빙)
```

### 3. 경제성 비교
```
초기 투자:
- Case 1: $50.6M (탱크 포함)
- Case 2-2: $16.1M (탱크 불필요)

20년 NPC:
- Case 1: $2.5~3.5B
- Case 2-2: $1.5~2.2B (30~40% 절감)
```

### 4. 의사결정 기준
| 상황 | 최적 선택 | 이유 |
|-----|---------|------|
| 초기 투자 최소화 | Case 2-2 | 67% 낮은 초기 투자 |
| 높은 유연성 필요 | Case 1 | 탱크의 즉각적 대응 |
| 거리 단축 중요 | Case 2-2 | 울산이 여수보다 16% 빠름 |
| 장기 경제성 | Case 2-2 | NPC 30~40% 절감 |
