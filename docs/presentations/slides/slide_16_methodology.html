Methodology Overview

Optimization Model: MILP (Mixed-Integer Linear Programming)

Minimize total Net Present Cost (NPC) across 20-year horizon (2030-2050) by determining optimal fleet size and equipment specifications

Decision Variables

  • Number of shuttles deployed annually (integer)
  • Shuttle size selection (500-5,000 m³ for Case 1; up to 50,000 m³ for Cases 2)
  • Pump capacity selection (400-2,400 m³/h)
  • Annual bunkering cycles

Cost Components Analyzed

  • CAPEX: Shuttle construction, bunkering equipment, storage tank
  • OPEX: Fuel consumption, crew costs, maintenance, cooling (storage)
  • Discount rate: 7% annual for NPV calculations

Solver

CBC (Coin-or Branch and Cut) via PuLP Python library with optimality gap < 0.1%
