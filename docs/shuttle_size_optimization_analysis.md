# 암모니아 벙커링 셔틀 크기 최적화 분석 보고서

## Optimal Shuttle Size Analysis for Ammonia Bunkering Operations

**버전**: 1.0
**작성일**: 2026-01-27
**프로젝트**: Green Corridor Ammonia Bunkering Optimization

---

## 1. Executive Summary

본 보고서는 암모니아 벙커링 셔틀의 최적 크기 결정에 관한 기술경제적 분석을 제시한다. 직관적으로 대형 셔틀이 규모의 경제를 통해 비용 효율적일 것으로 예상되지만, 실제 분석 결과 **중소형 셔틀(5,000-10,000 m³)이 대형 셔틀(50,000 m³)보다 경제적**임을 발견하였다.

**핵심 발견사항:**
- 50,000 m³ 셔틀은 선박당 시간을 52% 절감하지만
- CAPEX가 5.62배 증가하여
- 결과적으로 선박당 비용이 2.7배 높아짐

이러한 역설적 결과의 근본 원인은 **CAPEX 스케일링 지수(0.75)**가 **시간 효율성 향상폭(2.08배)**을 상쇄하지 못하기 때문이다.

## 2. 분석 배경 및 목적

### 2.1 문제 정의

부산항 그린 해운 회랑에서 암모니아 연료를 공급하기 위한 벙커링 셔틀의 최적 크기를 결정해야 한다. 셔틀 크기는 다음 요소에 영향을 미친다:

1. **운영 효율성**: 대형 셔틀은 한 번의 항해로 더 많은 선박에 급유 가능
2. **자본 비용(CAPEX)**: 대형 셔틀은 건조 비용이 높음
3. **운영 비용(OPEX)**: 연료비, 유지보수비 등

### 2.2 분석 대상 케이스

**Case 2-1: 여수 → 부산 (장거리 운송)**
- 편도 거리: 86 해리 (5.73시간 @ 15knot)
- 벙커링 수요: 5,000 m³/척
- 분석 셔틀 크기: 5,000 ~ 50,000 m³

## 3. 시간 구조 분석

### 3.1 1회 사이클 시간 구성

여수-부산 노선에서 셔틀 1회 왕복 사이클은 다음과 같이 구성된다:

```
Cycle_Duration = Shore_Loading + Basic_Cycle

Shore_Loading = 셔틀크기/1500 + 2.0h (육상 셋업)

Basic_Cycle = Travel_왕복(11.46h) + 부산항_진입출항(2.0h) + 선박당_작업(8.0h) × N
```

여기서:
- **Shore_Loading**: 육상 적재 시간 (셔틀 크기에 비례)
- **Travel_왕복**: 여수↔부산 항해 시간 (고정)
- **부산항_진입출항**: 부산항 영내 진입/출항 시간 (고정)
- **선박당_작업**: 영내 이동(3h) + 펌핑(5h) = 8h/척

### 3.2 셔틀 크기별 시간 분석

| 셔틀 크기 | Shore Loading | Basic Cycle | **Cycle 총시간** | 서빙 선박 |
|-----------|---------------|-------------|------------------|-----------|
| 5,000 m³ | 5.33h | 21.46h | **26.79h** | 1척 |
| 10,000 m³ | 8.67h | 29.46h | **38.13h** | 2척 |
| 15,000 m³ | 12.00h | 37.46h | **49.46h** | 3척 |
| 20,000 m³ | 15.33h | 45.46h | **60.79h** | 4척 |
| 50,000 m³ | 35.33h | 93.46h | **128.79h** | 10척 |

### 3.3 선박 1척당 시간 효율성

**핵심 지표: 선박당 소요 시간 = Cycle_Duration / 서빙_선박수**

| 셔틀 크기 | Cycle 시간 | 서빙 선박 | **선박당 시간** | 절약률 |
|-----------|------------|-----------|-----------------|--------|
| 5,000 m³ | 26.79h | 1척 | **26.79h** | 기준 |
| 10,000 m³ | 38.13h | 2척 | **19.07h** | -29% |
| 15,000 m³ | 49.46h | 3척 | **16.49h** | -38% |
| 20,000 m³ | 60.79h | 4척 | **15.20h** | -43% |
| 50,000 m³ | 128.79h | 10척 | **12.88h** | **-52%** |

**발견**: 50,000 m³ 셔틀은 선박당 시간을 52% 절감한다. 이동 시간(11.46h)이 10척에 분산되어 1.15h/척으로 감소하기 때문이다.

### 3.4 시간 구성 비율 분석

| 항목 | 5,000 m³ | 10,000 m³ | 50,000 m³ |
|------|----------|-----------|-----------|
| Shore Loading / 척 | 20% | 23% | 27% |
| 이동 (왕복) / 척 | **43%** | **30%** | **9%** |
| 셋업 (육상) / 척 | 7% | 5% | 2% |
| 영내 이동/접안 / 척 | 11% | 16% | 23% |
| 펌핑 / 척 | 19% | 26% | **39%** |

대형 셔틀일수록:
- 이동 시간 비율 감소 (43% → 9%)
- 펌핑 시간 비율 증가 (19% → 39%)

## 4. CAPEX 스케일링 분석

### 4.1 스케일링 법칙

선박 건조 비용은 일반적으로 다음 멱함수 관계를 따른다:

```
CAPEX = A × Size^α
```

여기서:
- A: 기준 비용 상수
- Size: 셔틀 용량 (m³)
- α: 스케일링 지수

### 4.2 현재 모델의 파라미터

```yaml
shuttle:
  ref_capex_usd: 61,500,000    # 기준 CAPEX (40,000 m³)
  ref_size_cbm: 40,000          # 기준 크기
  capex_scaling_exponent: 0.75  # 스케일링 지수
```

### 4.3 스케일링 지수 0.75의 의미

| 크기 비율 | CAPEX 비율 | 설명 |
|-----------|------------|------|
| 2x | 2^0.75 = 1.68x | 크기 2배 → CAPEX 1.68배 |
| 5x | 5^0.75 = 3.34x | 크기 5배 → CAPEX 3.34배 |
| 10x | 10^0.75 = 5.62x | 크기 10배 → CAPEX 5.62배 |

### 4.4 셔틀 크기별 CAPEX 비교 (5,000 m³ 기준)

| 셔틀 크기 | 크기 비율 | CAPEX 비율 (^0.75) | 상대 CAPEX |
|-----------|-----------|-------------------|------------|
| 5,000 m³ | 1.0x | 1.00x | 기준 |
| 10,000 m³ | 2.0x | 1.68x | +68% |
| 20,000 m³ | 4.0x | 2.83x | +183% |
| 50,000 m³ | 10.0x | **5.62x** | **+462%** |

## 5. 비용 효율성 분석

### 5.1 핵심 비교: 시간 효율 vs CAPEX

| 셔틀 크기 | 시간 효율 (선박당) | CAPEX 비율 | **비용/시간 효율** |
|-----------|-------------------|------------|-------------------|
| 5,000 m³ | 1.00x (26.79h) | 1.00x | 1.00 |
| 10,000 m³ | 0.71x (19.07h) | 1.68x | 1.19 (불리) |
| 50,000 m³ | 0.48x (12.88h) | 5.62x | **2.70 (매우 불리)** |

### 5.2 연간 서빙 용량 분석

연간 가동 시간 8,000시간 기준:

| 셔틀 크기 | 선박당 시간 | **연간 서빙 가능 선박** | CAPEX 비율 |
|-----------|-------------|-------------------------|------------|
| 5,000 m³ | 26.79h | 8000/26.79 = **299척** | 1.00x |
| 10,000 m³ | 19.07h | 8000/19.07 = **420척** | 1.68x |
| 50,000 m³ | 12.88h | 8000/12.88 = **621척** | 5.62x |

**용량 증가율**: 50,000 m³는 5,000 m³ 대비 621/299 = **2.08배** 용량

### 5.3 선박당 CAPEX 비용

```
선박당 CAPEX = CAPEX 비율 / 연간 서빙 용량

5,000 m³:  1.00 / 299 = 0.00335 (기준)
10,000 m³: 1.68 / 420 = 0.00400 (+19%)
50,000 m³: 5.62 / 621 = 0.00905 (+170%)
```

**결론**: 50,000 m³ 셔틀은 선박당 CAPEX가 **2.70배** 높다.

## 6. 수학적 분석: 최적 조건 도출

### 6.1 비용 효율성 조건

대형 셔틀이 비용 효율적이려면:

```
CAPEX 증가율 ≤ 용량 증가율
```

### 6.2 현재 모델 분석

5,000 m³ → 50,000 m³ (10배 증가) 시:

- **CAPEX 증가**: 10^0.75 = 5.62배
- **용량 증가**: 2.08배 (시간 효율성 기반)

```
5.62 > 2.08  →  대형 셔틀 불리
```

### 6.3 손익분기 스케일링 지수 계산

대형 셔틀이 동등하려면:

```
10^α = 2.08
α = log(2.08) / log(10) = 0.318
```

**손익분기점**: α = 0.318

| 스케일링 지수 | CAPEX 증가 (10x 크기) | 결과 |
|--------------|----------------------|------|
| 0.318 | 2.08x | 손익분기 |
| 0.50 | 3.16x | 불리 |
| **0.75 (현재)** | **5.62x** | **매우 불리** |
| 1.00 | 10.0x | 극히 불리 |

## 7. 100배 수요 시나리오 분석

### 7.1 수요 스케일링 영향

100배 수요(연간 60,000척 벙커링) 시에도 최적 셔틀 크기는 변하지 않았다.

| 시나리오 | 5,000 m³ LCO | 10,000 m³ LCO | 50,000 m³ LCO | 최적 |
|----------|-------------|---------------|---------------|------|
| 1x 수요 | 3.76 | 3.73 | 6.22 | 10,000 m³ |
| 10x 수요 | 3.68 | 3.59 | 5.80 | 10,000 m³ |
| 100x 수요 | 3.67 | **3.57** | 5.75 | **10,000 m³** |

### 7.2 왜 수요 증가가 대형 셔틀에 유리하지 않은가?

**구조적 문제**: CAPEX/용량 비율은 수요와 무관

```
어떤 수요에서도:
  - 5,000 m³ N척 필요 vs 50,000 m³ M척 필요
  - M = N × (299/621) = 0.48N (절반의 셔틀)
  - 하지만 CAPEX_50000 = 5.62 × CAPEX_5000
  - 총 비용: 0.48N × 5.62 = 2.70N > N

결과: 항상 2.70배 비용 불리
```

| 선종 | 용량 | 가격 | 단가 ($/m³) |

### 8.3 역산 스케일링 지수

LNG 벙커링 선박 vs LPG 캐리어:

```
(90/125) = (18,000/93,000)^α
0.72 = 0.194^α
α = ln(0.72) / ln(0.194) = 0.20
```

**역산 결과**: α ≈ 0.20 (현재 모델 0.75보다 훨씬 낮음)

## 9. 결론 및 시사점

### 9.1 주요 결론

1. **직관과 반대 결과**: 대형 셔틀이 항상 유리하지 않다
2. **근본 원인**: CAPEX 스케일링 지수(0.75)가 시간 효율성 향상(2.08배)을 상쇄하지 못함
3. **최적 크기**: 현재 파라미터에서 **10,000 m³**가 최적
4. **수요 무관**: 수요가 100배 증가해도 최적 크기는 변하지 않음

### 9.2 민감도 분석 요약

| 스케일링 지수 α | 최적 셔틀 크기 | 대형 셔틀 경쟁력 |
|----------------|----------------|------------------|
| 0.75 (현재) | 10,000 m³ | 없음 |
| 0.50 | ~20,000 m³ | 일부 |
| 0.32 이하 | 50,000 m³ | 완전 |

### 9.3 정책적 시사점

1. **인프라 계획**: 대형 셔틀 투자보다 중형 셔틀 다수 운영이 경제적
2. **조선 산업**: 소형 암모니아 벙커링 선박 시장에 기회
3. **향후 연구**: 실제 소형 셔틀 건조 비용 데이터 수집 필요

### 9.4 모델 개선 권고사항

1. **스케일링 지수 재검토**: 0.75 → 0.5-0.6 검토 필요
2. **크기별 차등 지수**: 소형/중형/대형 셔틀에 다른 지수 적용 고려
3. **실증 데이터 확보**: 실제 암모니아 벙커링 선박 건조 견적 수집

## 10. 부록: 수식 요약

### A. 시간 계산

```
Cycle_Duration = Shore_Loading + Basic_Cycle

Shore_Loading = Shuttle_Size / Shore_Pump_Rate + Shore_Setup_Time
              = Shuttle_Size / 1500 + 2.0  [hours]

Basic_Cycle = Travel_Roundtrip + Port_Entry_Exit + Per_Vessel_Work × N_vessels
            = 11.46 + 2.0 + 8.0 × N  [hours]

Per_Vessel_Time = Cycle_Duration / N_vessels
```

### B. CAPEX 계산

```
CAPEX(Size) = Ref_CAPEX × (Size / Ref_Size)^α

where:
  Ref_CAPEX = $61,500,000
  Ref_Size = 40,000 m³
  α = 0.75 (current model)
```

### C. 비용 효율성 지표

```
Cost_per_Vessel = CAPEX_Ratio / Annual_Capacity_Ratio

Annual_Capacity = Operating_Hours / Per_Vessel_Time
                = 8000 / (Cycle_Duration / N_vessels)

Cost_Efficiency_Ratio = (Size_Ratio)^α / Capacity_Ratio
```

### D. 손익분기 조건

```
대형 셔틀이 유리하려면:
  (Size_Ratio)^α ≤ Capacity_Ratio

손익분기 스케일링 지수:
  α_breakeven = log(Capacity_Ratio) / log(Size_Ratio)
              = log(2.08) / log(10)
              = 0.318
```
